# Contributing to DesktopSnap

First off, thank you for considering contributing to DesktopSnap! It's people like you that make it a better tool for everyone.

## How Can I Contribute?

### Reporting Bugs
*   Check if the bug has already been reported in the Issues section.
*   If not, open a new issue. Be as detailed as possible, including your Windows version, display configuration, and steps to reproduce.

### Suggesting Enhancements
*   Open an issue to discuss your idea before implementing it.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure your code follows the existing style.
4.  Issue that pull request!

## Local Development
To build the project locally, you need:
*   Visual Studio 2022 or VS Code.
*   .NET 8 SDK.
*   Windows App SDK (WinUI 3).

```bash
# Build via CLI
dotnet build
```

# 参与贡献 (中文)

## 我该如何参与？

### 报告 Bug
*   请先检查 Issues 中是否已有类似的反馈。
*   如果没有，欢迎提交新的 Issue。请尽量详细描述您的 Windows 版本、显示器配置以及复现步骤。

### 提交 PR
1.  Fork 本仓库并基于 `main` 分支创建您的功能分支。
2.  确保代码风格与现有项目一致。
3.  提交 Pull Request。

## 本地开发

*   Visual Studio 2022 或 VS Code。
*   .NET 8 SDK。
*   Windows App SDK (WinUI 3)。
